Ordered list with nested answers (15 items)

  1. What does HTML stand for?
    • Hyper Text Markup Language.
  2. Which tag creates a paragraph?
    • The <p> tag.
  3. Which tag creates a main heading?
    • The <h1> tag.
  4. Which tag inserts an image?
    • The <img> tag.
  5. Which attribute sets an image source?
    • The src attribute.
  6. Which attribute sets alternative text?
    • The alt attribute.
  7. Which tag creates a hyperlink?
    • The <a> tag.
  8. Which tag defines a table row?
    • The <tr> tag.
  9. Which tags define table header and cell?
    • <th> for header, <td> for data.
  10. What does colspan do?
    • It makes one cell span several columns.
  11. What does rowspan do?
    • It makes one cell span several rows.
  12. Which tag starts an unordered list?
    • The <ul> tag.
  13. Which tag starts an ordered list?
    • The <ol> tag.
  14. Which tag defines a list item?
    • The <li> tag.
  15. Why do we use comments in HTML?
    • To explain our code for ourselves or others.
